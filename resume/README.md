# Resume Directory

Place your resume files here (PDF or DOCX format).

## Supported Formats

- PDF (.pdf)
- Microsoft Word (.docx, .doc)

## File Naming

Use descriptive names like:
- `resume.pdf`
- `john_doe_resume_2024.pdf`

## Configuration

Update `resume_path` in `config.yaml` to point to your resume file.

Example:
```yaml
profile:
  resume_path: "./resume/resume.pdf"
```
